# MiTAC 線上教育訓練平台 - 操作手冊

本文件旨在提供 **一般使用者 (員工)** 與 **系統管理員** 對於本教育訓練平台的操作指南。

---

## 👥 一般使用者操作指南

### 1. 登入與註冊
系統採用 **員工編號 + 公司 Email** 雙重驗證機制。
- **首次使用 (註冊)**：
  1. 輸入您的 **員工編號** (4碼)、**中文姓名** 及 **公司 Email**。
  2. 點擊「開始學習 / 註冊」。
  3. 系統將自動建立您的帳號並登入。
- **後續登入**：
  1. 輸入您的 **員工編號**。
  2. 輸入當初註冊時的 **公司 Email** (需一致)。
  3. 點擊按鈕即可登入。

### 2. 選修課程
- 登入後，首頁會顯示目前 **開放中** 且 **您有權限觀看** 的課程。
- 每個課程卡片會顯示您的學習進度條 (例如：學習中 50% 或 已完成)。
- 點擊 **「進入課程」** 按鈕開始學習。

### 3.進行學習
課程內容包含影片與測驗：

- **🎥 影片單元**：
  - 系統會自動記錄您的觀看進度。
  - **重要限制**：一般學員 **無法手動拖曳進度條 (Seeking)** 跳轉影片，必須依序觀看。
  - 當觀看進度超過 **90%** 時，該單元將自動標記為「已完成」。
  - 系統支援「中斷點續看」，下次進入時會從上次結束的地方開始播放。

- **📝 測驗單元**：
  - 點擊測驗單元，會顯示測驗說明。
  - 點擊按鈕會開啟 Google Forms 測驗視窗。
  - **完成標記**：開啟測驗視窗後，系統會倒數 **10秒**，隨後「標記測驗已完成」按鈕會亮起。請在完成作答後手動點擊此按鈕以更新進度。

### 4. 查看個人紀錄
- 在導覽列點擊 **「我的學習紀錄」**。
- 可查看所有修課歷史，包含已完成與進行中的課程。
- 支援按年份收折檢視。

## 🛠️ 管理員後台操作指南

### 1. 管理員登入
- 在登入畫面下方點擊 **「管理員後台」** 連結。
- 輸入管理員帳密：
  - **帳號**：`admin`
  - **密碼**：`mitachr`

### 2. 課程管理 (Course Management)
進入後台預設為課程列表頁面：
- **新增課程**：點擊右上角 **「+ 新增課程」**，設定課程標題、開放日期、實際開課日期及課程單元 (影片連結或測驗連結)。
- **編輯課程**：點擊課程卡片上的 **「編輯」** 按鈕修改內容。
- **刪除課程**：
  - **單筆刪除**：點擊課程卡片上的 **「刪除」**。
  - **批次刪除**：勾選多個課程前的核取方塊，點擊上方出現的 **「🗑️ 刪除」** 按鈕。
- **課程排序與分類**：
  - 課程依 **年度** 與 **月份** 自動分組。
  - 可透過上方下拉選單切換排序方式：依「線上開放日期」、「實際課程日期」或「上架狀態 (ON AIR)」。
- **查看個別統計**：點擊 **「查看進度」** 可檢視該課程所有學員的完成狀況。

### 3. 學員管理 (User Management)
切換至「學員管理」頁籤：
- **新增學員**：可手動新增個別學員資料。
- **編輯/刪除**：可修改學員 Email、姓名，或刪除離職人員帳號。
- **批次刪除**：勾選多位學員後進行批次刪除。

### 4. 數據分析與匯出
- **📊 行為分析儀表板 (Behavioral Analytics)**：
  - 點擊 **「行為分析」** 頁籤。
  - **真實參與度 (TES)**：依據觀看時長與行為計算的品質分數。
  - **興趣回放 (Seek Backs)**：統計學員主動倒帶重看 (超過5秒) 的次數，可用於識別課程中的難點或重點。
  
- **📥 匯出學習紀錄 (CSV)**：
  - 在課程列表頁上方，點擊 **「📊 匯出紀錄」**。
  - **篩選年份**：選擇特定年份或所有年份的課程。
  - **選擇課程**：勾選欲匯出的課程標題。
  - **選擇欄位**：客製化匯出內容，包含：
    - 基本資料 (員編、姓名、課程名)
    - 學習狀態 (完成率、已完成單元數)
    - **詳細單元數據** (各單元是否完成、觀看進度 %、觀看次數)
  - 點擊 **「確定匯出」** 即可下載 CSV 報表。
